# Music Asset Readme

This folder contains music and sound assets.

# Background

Sound effects are stored in two files, sfx.ctl and sfx.tbl.

The .ctl file contains meta information about the sound effect. The .tbl file contains actual audio data.

Music uses three files. Two files are used to store instrument sounds, instruments.ctl and instruments.tbl. Track information is based on MIDI format, and is stored in a (compiled) soundbank file, music.sbk.
