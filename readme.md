This is a working Goldeneye 007 decompilation! 


The only real requirements should be 
```
sudo apt install binutils-mips-linux-gnu make
```
followed by
```
make
```


Due to a bug with gzip, the following is your friend for random sha1 fails
```
make clean && make
```
